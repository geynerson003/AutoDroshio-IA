Actúa como un arquitecto de software senior especializado en SaaS, inteligencia artificial, comercio electrónico y automatización de marketing digital.
Necesito diseñar y desarrollar un SaaS llamado "AutoDropship AI", cuyo objetivo es automatizar casi todo el proceso de dropshipping utilizando inteligencia artificial.
El sistema debe permitir que un usuario cree una tienda de dropshipping que automáticamente:
1. Investigue productos ganadores.
2. Analice tendencias del mercado.
3. Genere páginas de producto optimizadas para conversión.
4. Cree creativos publicitarios con IA.
5. Publique anuncios automáticamente en Meta Ads.
6. Publique productos en Facebook Marketplace.
7. Automatice el fulfillment con proveedores de dropshipping.
8. Analice el rendimiento de anuncios y optimice campañas automáticamente.
El sistema debe incluir los siguientes módulos:
Módulo 1: Product Research AI
- Analizar productos trending en TikTok, Amazon y AliExpress.
- Usar APIs o scraping para obtener productos con alta demanda.
- Evaluar métricas como:
  - volumen de búsqueda
  - engagement
  - número de ventas
  - margen de ganancia estimado
Módulo 2: AI Product Page Generator
- Generar automáticamente:
  - título optimizado
  - descripción persuasiva
  - beneficios del producto
  - FAQ
  - copy de ventas
- Generar imágenes o mejorar imágenes existentes con IA.
- Crear videos promocionales cortos para redes sociales.
Módulo 3: Store Automation
- Integración con Shopify o WooCommerce mediante API.
- Crear automáticamente páginas de producto.
- Sincronizar inventario y precios.
Módulo 4: AI Ads Generator
- Crear copies publicitarios.
- Generar variaciones de anuncios.
- Crear creativos (imagen o video) con IA.
- Generar diferentes audiencias objetivo.
Módulo 5: Ads Automation
- Integrarse con la API de Meta Ads.
- Crear campañas automáticamente.
- Crear ad sets.
- Subir creativos generados.
- Lanzar campañas con presupuestos definidos.
Módulo 6: Auto Fulfillment
- Integrarse con proveedores de dropshipping como CJ Dropshipping o AliExpress.
- Cuando se genere una venta, crear automáticamente la orden al proveedor.
- Obtener número de tracking.
Módulo 7: AI Optimization Engine
- Analizar rendimiento de anuncios.
- Detectar anuncios ganadores.
- Escalar presupuesto automáticamente.
- Pausar anuncios con bajo rendimiento.
Módulo 8: Dashboard SaaS
- Panel para el usuario donde pueda ver:
  - ventas
  - productos activos
  - campañas publicitarias
  - ganancias
  - métricas clave
Tecnologías recomendadas:
Backend:
- Spring Boot o Node.js
- PostgreSQL
- Redis para caché
IA:
- OpenAI API
- modelos para generación de copy
- modelos para análisis de tendencias
Scraping y análisis:
- Python
- BeautifulSoup
- Selenium
Frontend:
- React
- Next.js
- TailwindCSS
Infraestructura:
- Docker
- Kubernetes
- AWS o Azure
Seguridad:
- autenticación JWT
- multi-tenant SaaS architecture
- rate limiting
- protección contra scraping abusivo
